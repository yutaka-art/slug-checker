GitHub 利用可否チェック

⚠️ 重要な注意事項

Enterprise Slugの正確な判定のため、以下の手順を推奨します：

  • GitHubからサインアウトしてから本ページを使用してください
  • または、シークレット/プライベートブラウザモードで本ページを開いてください
  • GitHubにサインインしている状態では、Enterprise Slugの有効性を正確に確認できません

※ GitHubユーザー名チェックは、サインイン状態に関係なく正常に動作します。

GitHub ユーザー名チェック

英数字とハイフンのみ使用可能（先頭・末尾のハイフン不可、最大39文字）

Enterprise Slug 存在確認

重要: Enterprise Slugの確実な判定は技術的制約により自動化が困難です。
下の手順で手動確認を行ってください：
  1. GitHubからサインアウト（またはシークレットモードで実行）
  2. 「ページを開く」ボタンをクリック
  3. 開いたページの結果を確認
📋 判定方法:
• 404 Not Found → ✅ そのslugは利用可能です
• ページが表示される → ❌ そのslugは既に使用中です
• ログイン画面への誘導 → ❌ そのslugは既に使用中です
• アクセス拒否 → ⚠️ 判定困難（再試行推奨）
